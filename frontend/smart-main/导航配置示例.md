# 导航配置示例

## 当前配置（默认 - 所有链接在当前页面跳转）

```typescript
export const PORTAL_NAV_ITEMS: NavItem[] = [
  { label: '首页', path: '/' },
  { label: '产链直销', path: '/mall' },
  { label: '助农金融', path: '/finance' },
  { label: '专家智库', path: '/knowledge' },
  { label: '用户社区', path: '/community' },
  { label: '帮助中心', path: '/help' },
];
```

---

## 如果你想让所有主要模块在新标签页打开

修改 `frontend/smart-main/constants.ts` 文件：

```typescript
export const PORTAL_NAV_ITEMS: NavItem[] = [
  { label: '首页', path: '/' },
  { label: '产链直销', path: '/mall', openInNewTab: true },
  { label: '助农金融', path: '/finance', openInNewTab: true },
  { label: '专家智库', path: '/knowledge', openInNewTab: true },
  { label: '用户社区', path: '/community', openInNewTab: true },
  { label: '帮助中心', path: '/help' },
];
```

**效果**: 点击"产链直销"、"助农金融"、"专家智库"、"用户社区"时会在新标签页打开

---

## 推荐配置（只让帮助中心在新标签页打开）

```typescript
export const PORTAL_NAV_ITEMS: NavItem[] = [
  { label: '首页', path: '/' },
  { label: '产链直销', path: '/mall' },
  { label: '助农金融', path: '/finance' },
  { label: '专家智库', path: '/knowledge' },
  { label: '用户社区', path: '/community' },
  { label: '帮助中心', path: '/help', openInNewTab: true },
];
```

**原因**: 
- 帮助文档通常需要对照查看
- 其他模块保持流畅的单页应用体验

---

## 使用说明

1. 打开 `frontend/smart-main/constants.ts`
2. 找到 `PORTAL_NAV_ITEMS` 数组
3. 在需要新标签页打开的导航项后添加 `, openInNewTab: true`
4. 保存文件
5. 刷新浏览器查看效果

详细说明请查看 [导航链接配置说明.md](./导航链接配置说明.md)
