用户社区模块详细设计
模块代码: COM (Community)
主色调: 活力橙 (#FF9800 - 互动/热情), 基础绿 (#4CAF50 - 农业基因)
访问路径: /community
核心理念: 互助、分享、共赢。
1. 社区首页 (Community Feed)
页面ID: COM-01
访问路径: /community (门户布局)
核心目标: 信息流分发，展示社区热度，引导用户参与。
1.1 布局架构
采用经典的 三栏布局 (Left-Center-Right)，这是社区类产品最高效的布局。

1.2 核心组件设计
[C-Editor-Trigger] 发帖触发器
样式: 白色卡片，Padding 16px。
内容: 用户头像 + 灰色输入框 (假输入框)。
交互: 点击输入框，弹出 [COM-04] 发帖弹窗。
快捷按钮: [📷 发图片] [📹 发视频] [❓ 提问]。
[C-Post-Card] 帖子卡片
容器: 卡片格式二，Margin-bottom 16px。
Header:
Avatar (40px) + Nickname (加粗) + Time (灰色 "2小时前")。
User Tag: [金牌农户] [农业专家]。
More Action: 右上角 "..." (举报/收藏)。
Content:
Title: 18px 加粗 (如果有)。
Text: 15px，行高 1.6，最多显示 3 行，超长显示 "...展开"。
Media: 图片九宫格 (Grid) 或 单个视频播放器。
Topic Tag: #苹果种植技术# (蓝色胶囊样式)。
Footer (Interaction Bar):
Flex 均分 3 列。
Btn 1: 👍 点赞 (激活变橙色，带数字)。
Btn 2: 💬 评论 (带数字)。
Btn 3: ↗ 分享。
[C-Hot] 热门话题榜
容器: 白色卡片。
List:
#2025粮补政策# (热度 1.2w 🔥)
#山东大葱涨价# (热度 8k)
#草莓育苗技巧# (热度 5k)
2. 话题/圈子详情页 (Topic Detail)
页面ID: COM-02
访问路径: /community/topic/:id
核心目标: 聚合垂直内容，形成兴趣圈层。
2.1 布局架构
顶部 Banner:
背景: 话题相关大图 (高度 150px)。
内容: 话题名称 (#苹果种植交流#) + 参与人数 + "加入圈子" 按钮。
下方布局: 与首页类似，但去除了左侧导航，侧重显示该话题下的精华帖。
3. 帖子详情页 (Post Detail)
页面ID: COM-03
访问路径: /community/post/:id
核心目标: 深度阅读与讨论。
3.1 组件设计
[C-Post-Main] 正文区
样式: 类似于文章详情，但更轻量。
关联: 底部增加 "相关农资推荐" (如果帖子讨论了病虫害，推荐相关药剂)。
[C-Comment-Area] 评论区
输入框: 底部固定 (Sticky Bottom) 或 文末显示。
评论列表:
一级评论: 头像 + 内容 + 操作栏 (点赞/回复)。
二级回复 (楼中楼): 灰色背景块包裹，显示 "作者回复: xxxx"。
神评论/置顶: 带有 "🔥 热门" 或 "💡 专家回答" 标识的评论排在最前。
4. 发帖/提问弹窗 (Create Post Modal)
页面ID: COM-04
交互: 模态框 (Modal) 或 抽屉 (Drawer)。
4.1 界面元素
Header: "发布动态" | "发布按钮" (Solid Green)。
Body:
Textarea: min-height: 150px，Placeholder "分享你的农业趣事或种植难题..."。
Media Uploader: 图片/视频上传，支持拖拽。
Tool Bar:
# 话题 (选择或新建)。
@ 提醒谁看。
📍 所在位置 (显示 "山东·烟台")。
Type Switch: 底部切换 [发动态] / [提问题]。如果是提问题，增加 "悬赏积分" 选项。

6. 内容审核后台 (Admin Moderation)
页面ID: A-05 (管理后台)
核心目标: 确保社区内容合规，过滤广告和违法信息。
6.1 审核列表
列: 内容预览 | 发布人 | 发布时间 | 机器审核结果 (通过/疑似违规/违规) | 操作。
机器审核: 接入阿里云/腾讯云的内容安全 API，自动标记色情/暴恐/涉政内容。
操作: [通过] [删除] [封号]。
