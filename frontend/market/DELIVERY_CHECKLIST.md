## 📝 使用说明

### 第一次使用

1. **安装依赖**
   ```bash
   cd frontend/market
   npm install
   ```

2. **启动后端服务**
   ```bash
   cd backend/market
   node index.js
   ```

3. **启动前端服务**
   ```bash
   cd frontend/market
   npm run dev
   ```

4. **访问应用**
   - 打开浏览器访问：http://localhost:5175
   - 注册新账户或使用测试账户登录

## 🔧 常见问题

### 1. 依赖安装失败
```bash
npm cache clean --force
rm -rf node_modules package-lock.json
npm install
```

### 2. 端口被占用
修改 `vite.config.ts` 中的端口配置

### 3. API 请求失败
- 确保后端服务已启动
- 检查后端服务端口（默认 3003）
- 查看浏览器控制台错误信息

### 4. 图片无法显示
- 检查图片 URL 是否正确
- 确保图片资源可访问
