## 📋 使用说明

### 1. 安装依赖
```bash
cd frontend/community
npm install
```

### 2. 启动开发服务器
```bash
npm run dev
# 或
start-dev.bat
```

### 3. 访问应用
```
http://localhost:5174
```

### 4. 构建生产版本
```bash
npm run build
```

## ⚠️ 注意事项

### 依赖安装
- 首次使用必须运行 `npm install`
- 确保 Node.js >= 18
- 确保 npm >= 9

### 后端服务
- 需要后端服务运行在 http://localhost:3003
- 确保后端数据库已初始化
- 确保后端 API 可访问

### 开发环境
- 前端端口：5174
- 后端端口：3003
- 代理配置在 vite.config.ts

### TypeScript 错误
- 安装依赖前会有类型错误，这是正常的
- 运行 `npm install` 后错误会消失
- 如仍有错误，检查 tsconfig.json 配置
