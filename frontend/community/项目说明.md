# 智农链社区前端模块

## 项目概述

这是智农链平台的社区模块前端应用，采用现代化的技术栈和莫兰迪色系设计风格，为用户提供优雅、易用的社区交流体验。

## 设计理念

### 配色方案（莫兰迪色系）

**主色调 - 深绿**
- 主色：#5F7367（莫兰迪深绿）
- 浅绿：#7A8E82
- 深绿：#4A5B52
- 背景：#F5F7F6

**辅助色**
- 砖红：#A67C7C（用于警告、错误提示）
- 沙蓝：#7B8FA3（用于信息提示）
- 暗金：#B8A57A（用于悬赏、特殊标记）

**中性色**
- 白色：#FFFFFF
- 背景：#F8F9F8
- 边框：#E5E8E6
- 文字：#3D4A42
- 次要文字：#6B7870
- 辅助文字：#9BA39E

### 设计特点

1. **简洁优雅**：采用卡片式布局，留白充足，视觉舒适
2. **色彩柔和**：莫兰迪色系营造温和、专业的氛围
3. **交互友好**：按钮、链接等交互元素有明确的视觉反馈
4. **响应式设计**：适配不同屏幕尺寸

## 核心功能

### 1. 内容浏览与筛选
- **首页 Feed 流**：展示社区最新、最热内容
- **分类导航**：支持多级分类筛选
- **排序方式**：热门、最新、推荐
- **内容卡片**：显示标题、作者、封面、统计数据

### 2. 内容详情与互动
- **完整内容展示**：支持富文本内容
- **点赞收藏**：一键互动
- **评论系统**：支持评论和回复
- **悬赏问答**：显示悬赏金额和解决状态

### 3. 内容发布
- **三种类型**：
  - 经验分享：分享种植、销售经验
  - 求助：寻求帮助和建议
  - 问题咨询：发布悬赏问题
- **分类选择**：多级分类树形选择
- **标签系统**：智能标签搜索和添加
- **悬赏设置**：问答类型可设置悬赏金额

### 4. 问答系统
- **悬赏发布**：提问者设置悬赏金额
- **答案采纳**：提问者可采纳最佳答案
- **奖励发放**：采纳后自动发放奖励
- **状态标识**：清晰显示问题解决状态

### 5. 社交功能
- **用户关注**：关注感兴趣的用户
- **互相关注标识**：显示互关状态
- **粉丝系统**：查看关注者
- **黑名单**：屏蔽不良用户

## 技术实现

### 状态管理
使用 Zustand 进行轻量级状态管理：
- 用户登录状态
- 当前用户信息

### HTTP 请求
使用 Axios 封装统一的请求处理：
- 自动添加 Authorization 头
- 统一错误处理
- 401 自动跳转登录

### 路由设计
```
/                    - 首页（内容列表）
/content/:id         - 内容详情
/publish             - 发布内容
```

### API 接口模块化
```
api/
├── content.ts       - 内容 CRUD、点赞、收藏、评论
├── follow.ts        - 关注、取关、黑名单
├── qa.ts            - 采纳答案、取消采纳
└── report.ts        - 举报功能
```

## 组件设计

### 布局组件
- **Header**：顶部导航栏，包含 Logo、导航链接、用户信息
- **Sidebar**：侧边栏，展示分类树

### 业务组件
- **ContentCard**：内容卡片，展示内容摘要信息
- **CommentItem**：评论项，支持嵌套回复

### 页面组件
- **Home**：首页，内容列表 + 侧边栏
- **ContentDetail**：内容详情页
- **Publish**：发布页面

## 开发规范

### 代码风格
- 使用 TypeScript 严格模式
- 组件使用函数式组件 + Hooks
- 样式使用内联对象（便于主题定制）

### 类型定义
所有 API 响应、组件 Props 都有完整的类型定义

### 错误处理
- API 请求统一 try-catch
- 用户友好的错误提示
- 控制台输出详细错误信息

## 部署说明

### 开发环境
```bash
cd frontend/community
npm install
npm run dev
```

### 生产构建
```bash
npm run build
```

构建产物在 `dist` 目录，可部署到任何静态服务器。

### 环境变量
可在 `.env` 文件中配置：
```
VITE_API_BASE_URL=http://localhost:3003
```
