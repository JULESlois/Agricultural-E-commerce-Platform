# 社区主页更新说明

## 更新内容

参考 bilibili 社区，重新设计了社区主页的三栏布局：

### 1. 左侧栏 - 用户信息卡片
- 显示用户头像（支持自定义头像或首字母占位符）
- 显示用户名
- 显示三个统计数据：
  - 关注数
  - 粉丝数
  - 发布的帖子数

文件：`src/components/Layout/UserSidebar.tsx`

### 2. 中间栏 - 帖子列表
- 保留原有的排序功能（热门/最新/推荐）
- 优化了排序标签的样式，添加了 emoji 图标
- 帖子卡片使用半透明背景和毛玻璃效果

文件：`src/pages/Home.tsx`

### 3. 右侧栏 - 热门话题和商城入口
- 热门话题列表：
  - 显示排名（前三名高亮）
  - 显示话题标题
  - 显示热度值
- 商城入口按钮（暂未实现跳转逻辑）

文件：`src/components/Layout/RightSidebar.tsx`

### 4. 背景图支持
- 页面支持背景图，使用视差滚动效果
- 所有卡片使用半透明背景 + 毛玻璃效果（backdrop-filter: blur）
- 背景图路径：`src/assets/background.jpg`

## 样式特点

1. **毛玻璃效果**：所有卡片使用 `backdrop-filter: blur(10px)` 实现毛玻璃效果
2. **半透明背景**：使用 `rgba(255, 255, 255, 0.95)` 实现半透明白色背景
3. **阴影效果**：使用 `box-shadow: 0 2px 8px rgba(0, 0, 0, 0.06)` 增加层次感
4. **圆角设计**：统一使用 12px 圆角
5. **响应式布局**：使用 flex 布局，支持不同屏幕尺寸

## 布局尺寸

- 左侧栏：240px（固定宽度）
- 中间栏：自适应（flex: 1）
- 右侧栏：280px（固定宽度）
- 容器最大宽度：1400px
- 栏间距：20px
