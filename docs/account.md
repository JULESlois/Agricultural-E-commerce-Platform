# 账户中心与个人资料管理手册

**版本**: 1.0
**最后更新**: 2023-10-27

---

## 1. 概述

本手册旨在为平台用户提供关于“账户中心”模块的全面操作指南。账户中心是您管理个人信息、保障账户安全以及个性化平台体验的核心区域。请仔细阅读本手册，以确保您能充分利用所有功能。

---

## 2. 个人资料管理

个人资料是您在平台上的数字身份象征，保持其准确性有助于提升互动体验。

- **访问路径**：登录后，点击页面右上角的头像，在下拉菜单中选择“个人中心” -> “账户设置”。

- **可编辑字段**：
  - **头像 (Avatar)**：上传一张代表您个人或品牌的图片。支持`.jpg`, `.png`格式，推荐尺寸为200x200像素。
  - **昵称 (Nickname)**：您在社区中显示的名称，长度限制为4-20个字符。
  - **个人简介 (Bio)**：一段简短的自我介绍，将显示在您的个人主页，帮助其他用户了解您。

---

## 3. 账户安全设置

我们高度重视您的账户安全，并提供多层次的安全防护措施。

- **访问路径**：“个人中心” -> “安全设置”。

### 3.1. 密码管理

- **修改密码**：为了保障账户安全，我们建议您每隔90天更换一次密码。新密码必须满足以下要求：
  - 长度至少为8个字符。
  - 必须包含大写字母、小写字母、数字和特殊符号（如`!@#$%^&*`）中的至少三种。
- **忘记密码**：如果您忘记了密码，可以在登录页面点击“忘记密码”，通过您绑定的手机号码或电子邮箱重置密码。

### 3.2. 双重认证 (Two-Factor Authentication, 2FA)

为提供最高级别的安全保障，我们强烈建议您启用双重认证。

- **启用流程**：
  1. 在“安全设置”页面，选择“启用双重认证”。
  2. 使用您的手机（如Google Authenticator, Authy）扫描屏幕上显示的二维码。
  3. 输入验证器应用生成的6位动态验证码以完成绑定。
- **登录流程**：启用后，每次登录时，在输入密码之后，您都需要提供来自验证器应用的动态验证码。

### 3.3. 绑定验证方式

- **手机号码绑定**：用于接收登录验证码、重要通知以及作为密码重置的主要途径。
- **电子邮箱绑定**：作为备用验证方式，并用于接收平台资讯和账单。

## 4. 隐私与数据管理

我们尊重并保护您的个人隐私。

- **访问路径**：“个人中心” -> “隐私设置”。

- **可控选项**：
  - **个人主页可见性**：您可以设置您的个人主页对所有人可见、仅粉丝可见或仅自己可见。
  - **动态可见范围**：在发布每条动态时，您可以单独设置其为公开、好友圈或私密。
  - **数据导出**：您有权申请导出您在平台上的个人数据副本，申请将在7个工作日内处理。

---

## 5. 通知管理

您可以自定义希望接收的平台通知类型，以避免不必要的信息干扰。

- **访问路径**：“个人中心” -> “通知设置”。

- **通知渠道**：
  - **站内信**：平台内部的通知消息。
  - **电子邮件**：发送至您绑定的邮箱。
  - **短信**：发送至您绑定的手机号码（仅限重要安全通知）。

- **可配置的通知类型**：
  - **互动通知**：如新的粉丝、点赞、评论和@提及。
  - **系统通知**：如平台维护公告、政策更新。
  - **营销与推广**：如新品推荐、优惠活动（可选择退订）。
