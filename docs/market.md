# 商城使用指南

本指南将帮助您了解如何在智慧商城中进行购物、订单管理和售后处理。

### 浏览与搜索商品

- 您可以通过顶部分类导航栏浏览不同类别的商品。
- 使用顶部的搜索框，输入关键词可以快速查找您需要的商品。

### 下单与支付

1.  在商品详情页，选择规格和数量，点击“加入购物车”或“立即购买”。
2.  在购物车页面，确认商品信息后，点击“去结算”。
3.  填写或选择您的收货地址，选择支付方式，确认无误后提交订单并完成支付。

### 订单管理

- 在“个人中心” -> “我的订单”中，您可以查看所有订单的状态。
- 您可以对订单进行“确认收货”、“申请售后”等操作。

### 售后服务

- 如果您对收到的商品不满意，可以在订单详情页点击“申请售后”。
- 填写售后申请表单，我们的客服人员会尽快与您联系处理。
